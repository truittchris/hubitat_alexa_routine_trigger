# Security Policy

This project is provided as-is. If you believe you have found a security issue, please report it privately.

## How to report

1. Open a GitHub issue and state clearly that you need a private reporting channel, or
2. Use the contact options on https://christruitt.com.

## What to include

- Hubitat platform version and model
- Steps to reproduce
- Relevant logs (redact personal information)

## Response expectations

Best effort will be made to review and address issues, but there is no guaranteed response time.
